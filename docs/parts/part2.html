XAM205 Navigation patterns in Xamarin.Forms

Exercise 2: Customize the navigation bar (XAM205)

In this exercise you'll change the colors of the navigation bar and control the text displayed when navigating back to a previous page.

This exercise is a continuation of the previous exercise, you may use your existing solution or open the completed solution provided for Exercise 1.

To complete the exercise, you will need Visual Studio for Windows or macOS with the Xamarin development tools installed. You will also need either an emulator/simulator or a device to run the exercise on. Please see the setup page if you need help installing the Xamarin development environment.

Open the starter solution

Open the starter solution from the Exercise 2 > Start folder in your copy of the cloned or downloaded course materials in either Visual Studio on Windows or Visual Studio for Mac.

Change the navigation bar color

  1. Open App.xaml.cs.
  2. In the constructor, create a local variable named navPage and assign it to the instance of NavigationPage.
  3. Assign a Color to navPage.BarBackgroundColor to change the color of the navigation bar.
  4. Assign a Color to navPage.BarTextColor to change the color of the navigation bar text.
  5. Assign the navPage variable to the Page's MainPage property.
public App ()
{
   InitializeComponent();
   var navPage = new NavigationPage(new Astronomy.MainPage());

   navPage.BarBackgroundColor = Color.Black;
   navPage.BarTextColor = Color.White;

   MainPage = navPage;
}

Optimize the back button text

The back button text is auto-populated to the title of the previous page. You'll provide alternate text for the pages with longer titles.

  1. Open MainPage.xaml.cs.

  2. In the constructor, after the call to InitializeComponent, call NavigationPage.SetBackButtonTitle passing in "Menu" for the back button text.

    public MainPage()
{
   InitializeComponent();

   NavigationPage.SetBackButtonTitle(this, "Menu");
   ...
}
  3. Perform the same steps in AstronomicalBodies.xaml.cs - the completed solution sets the back button text to "Back".
  4. Run the application and navigate between pages - notice the navigation bar colors have been updated and the back button text has been changed.

Exercise summary

In this exercise, you customized the colors and back button text of the navigation bar. This allows you to match style/theme preferences for your application and ensure the back button text is a reasonable length.
